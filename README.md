# Тестовое задание от soft-artel на Angular
Выполнено в полном объеме

## Как запускать
Клонирует репозиторий
```bash
git clone https://github.com/webbomj/soft-artel-test.git .
```
Устанавливаете зависимости
```bash
npm i
```
Запускаете фейковый сервер
```bash
npx json-server db.json 
```

Запускаете сайт в dev режиме с помощью NG
```bash
ng serve
```

## Использовалось
Angular 17 (signal, new flow)
NgRx + effect 17
Json-server
